# Reference Standard Selection

Toward Standardization uses declared reference standards. A reference standard is not assumed to be universal. It should be selected, documented, and reviewed for the specific biological model being benchmarked.

For the lung example, the declared reference is healthy human distal lung tissue.

Future extensions may use different references depending on the biological question, such as:

- healthy human tissue
- disease-specific tissue
- clinical data
- developmental reference datasets
- community-selected benchmark datasets

Community input is needed to define appropriate reference standards for each biological system.
